# Upcoming Core Features

The roadmap for Identity, Authentication, and User Management.

- [ ] **Role-Based Access Control (RBAC)**: Distinguish between regular players and League Admins.
- [ ] **Notification System**: Webhooks for Slack/Discord integration on match results.
- [ ] **Enhanced Profiles**: Allow users to set a nickname, bio, and custom preferences.
- [ ] **User Search**: fuzzy search to easily find colleagues by name or email.
- [ ] **Audit Logs**: Track sensitive actions (like admin overrides) for security.
- [ ] **Social Features**: Friend requests or "Rival" flagging.
